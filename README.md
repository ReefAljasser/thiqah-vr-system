# ThiqaVR System

ThiqaVR is a Virtual Reality (VR)-based public speaking exposure system designed to help participants manage public speaking anxiety through immersive VR environments, physiological monitoring, and machine learning analysis.

The system combines VR simulation, participant physiological data collection, machine learning classification, and a web dashboard for session and participant management.
